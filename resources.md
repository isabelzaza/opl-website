<h2>Research Tests</h2>

Downloadable materials for tests designed and validated by the Vanderbilt Object Perception Lab for use in individual differences research. These tests are freely available to promote research in this area.

**Please cite the reference provided for each test.**

<h2>Tests Using jsPsych</h2>

Tests to measure visual and auditory object recognition ability using jsPsych ([GitHub](https://github.com/OPLabVanderbilt/Ojs){:target="_blank"})

<h3>Vanderbilt Holistic Processing Tests for Novel Objects (VHPT-NO)</h3>

Designed to measure holistic processing for three categories of novel objects (asymmetric Greebles, Sheinbugs and one type of Ziggerins).

**Reference:** Chua, K.-W., & Gauthier, I. (2019). Domain-Specific Experience Determines Individual Differences in Holistic Processing. *Journal of Experimental Psychology: General.*
<span style="white-space: nowrap;">[pdf](https://www.dropbox.com/s/da2zxrdgciufkgh/ChuaGauthier_2019_JEPG.pdf?dl=0){:target="_blank"} | [materials](https://figshare.com/articles/VHPT_NO_zip/8345597#){:target="_blank"}</span>

<h3>Vanderbilt Chest Radiograph Test (VCRT)</h3>

A reliable measure of individual differences in perceptual abilities for radiograph-related decision-making in novices.

**Reference:** Sunday, M. A., Donnelly, E., & Gauthier, I. (2017). Individual differences in perceptual abilities in medical imaging: the Vanderbilt Chest Radiograph Test. *Cognitive Research: Principles and Implications, 2*(1), 36. [materials](https://figshare.com/articles/VCRT/5827770){:target="_blank"}

<h3>Vanderbilt Car Matching Test / Vanderbilt Face Matching Tests (VCMT/VFMT)</h3>

Tests designed to measure face and object recognition without any stimulus repetition.

**Reference:** Sunday, M. A., Lee, W.-Y., & Gauthier, I. (in press). Age-related differential item functioning in tests of face and car recognition ability. *Journal of Vision.* [materials](https://doi.org/10.6084/m9.figshare.5709682.v1){:target="_blank"}

<h3>Novel Object Memory Tests (NOMTs)</h3>

Three reliable tests to measure object recognition ability with novel objects.

**Reference:** Richler, J.J., Wilmer, J.B., Gauthier, I. (2017). General object recognition is specific: evidence from novel and familiar objects. *Cognition,* 166: 42-55.
<span style="white-space: nowrap;">[pdf](https://www.dropbox.com/s/66aa7jj1xae0tjq/RiWiGa_Cognition2017.pdf?dl=0){:target="_blank"} | [materials](https://figshare.com/s/e2cb4e01762580e46391){:target="_blank"}</span>

<h3>Vanderbilt Holistic Processing Test - Faces (VHPT-F 2.1)</h3>

A reliable test of holistic processing of faces, operationalized as a failure of selective attention.

**Reference:** Richler, J.J., Floyd, R.J., & Gauthier, I. (2014). The Vanderbilt Holistic Face Processing Test: A short and reliable measure of holistic face processing. *Journal of Vision*, 14, 1-14.
<span style="white-space: nowrap;">[pdf](http://www.journalofvision.org/content/14/11/10){:target="_blank"} | [materials](https://figshare.com/articles/VHFPT_2_0_zip/5432152){:target="_blank"}</span>

<h2>Vanderbilt Expertise Test (VET)</h2>

<h3>VET 1.0</h3>

The Vanderbilt Expertise Test is a battery of tests for object recognition abilities that uses a learning task.

**Reference:** McGugin, R.W., Richler, J.J., Herzmann, G., Speegle, M. & Gauthier, I. (2012). The Vanderbilt Expertise Test Reveals Domain-General and Domain-Specific Sex Effects in Object Recognition. *Vision Research*, 69:10-22.
<span style="white-space: nowrap;">[pdf](http://gauthier.psy.vanderbilt.edu/wp-content/uploads/2012/03/McRiHeGaSp.pdf){:target="_blank"} | [materials](https://figshare.com/s/2938870aa46ecfbab567){:target="_blank"}</span>

<h3>VET 2.0</h3>

An improved version with different object categories.

**References:**
- Van Gulick, A.B. (2014) Measurement of Semantic Knowledge and Its Contribution to Object Recognition Performance. Doctoral Dissertation, Vanderbilt University. [pdf](http://etd.library.vanderbilt.edu/available/etd-05222014-170633/unrestricted/VanGulick.pdf){:target="_blank"}
- Van Gulick, A., & Gauthier, I. (2014). Measurement of semantic knowledge of object categories: Creating the Semantic Vanderbilt Expertise Test (SVET). *Journal of Vision*, 14(10), 821.
<span style="white-space: nowrap;">[link](http://www.journalofvision.org/content/14/10/821.short){:target="_blank"} | [materials](https://figshare.com/s/dbfc860692c80133183d){:target="_blank"}</span>

<h3>VET House from Richler et al. (2019)</h3>

These trials are provided in [a PPT file](https://www.dropbox.com/s/ll51sdevdo6whyn/VET_House_trials.pptx?dl=0){:target="_blank"} that can be used to save individual images for all trials (see notes for details).

<h2>Semantic Vanderbilt Expertise Test (SVET 1.0)</h2>

A measure of semantic knowledge based on nomenclature, for the same categories as the VET 2.0.

**Reference:** Van Gulick, A. E., McGugin, R. W., & Gauthier, I. (2016). Measuring nonvisual knowledge about object categories: the semantic vanderbilt expertise test. *Behavior research methods*, 48(3), 1178-1196.
<span style="white-space: nowrap;">[pdf](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4754162/){:target="_blank"} | [materials](https://figshare.com/s/7e8269ba93c85d069893){:target="_blank"}</span>
